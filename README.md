## 重点募集カテゴリー

### 🔴 最優先
- **税理士** — 相続チーム完成の最後の1ピース
- **経営コンサル（補助金含む）** — ビジネスDXチームの核
- **映像制作/動画編集** — クリエイティブチーム強化

### 🟡 高優先
- 葬儀関係 / 広告運用 / インテリアデザイナー / ヨガ・ピラティス

### 🟢 推奨
- ドレス・衣装 / コピーライター / パーソナルトレーナー / 引越し / 店舗設計 / ネイル
